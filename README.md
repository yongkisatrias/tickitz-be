# Tickitz Backend

Backend for tickitz build with express

## Installation

```bash
# Setup the db using postgres
# Configure the dot env

# install the dependency
npm install

# run
npm run start

# run dev
npm run dev

```

## Package

1. express
2. postgres
3. bcrypt
4. cors
5. dotenv
6. helmet
7. jsonwebtoken
8. node-input-validator
